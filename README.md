# 📊 Sales Performance Dashboard — Excel

An interactive sales dashboard built in Microsoft Excel using
real sales data across regions, products, and sales reps.

## 📁 Dataset
- 1,500 rows of sales data
- Columns: Date, Region, Product, Salesperson,
  TotalPrice, Quantity, Discount, CustomerType

## 📌 Dashboard Features
- ✅ KPI Cards — Total Revenue, Orders, Avg Order Value, Units Sold
- ✅ Monthly Revenue Trend (Line Chart)
- ✅ Quarterly Revenue Comparison (Column Chart)
- ✅ Sales by Region & Product (Bar Chart)
- ✅ Sales Rep Performance (Bar Chart)
- ✅ Interactive Slicers — Region, Product, Salesperson, Quarter

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
